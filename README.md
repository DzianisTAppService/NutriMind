# NutriMind

Кликабельный прототип приложения для благополучия, которое объединяет три сферы — **еду**, **спорт** и **мысли** — в одну осознанную практику. Без подсчёта калорий, таймеров и навязанного позитива.

Подробное описание идеи, механик и открытых вопросов — в [`CONCEPT.md`](./CONCEPT.md).

## Стек

- React 19 + TypeScript
- Vite
- React Router
- SCSS-модули
- lucide-react (иконки)

Данные мок-овые, бэкенда нет — это фронтенд-прототип.

## Запуск

```bash
npm install
npm run dev
```

Откроется на http://localhost:5173/

## Сборка

```bash
npm run build
npm run preview
```

## Что внутри

- **Онбординг** — выбор приоритетной сферы и наставника
- **Сегодня** — интерактивный «Треугольник дня» + лента событий
- **Практики** — библиотека микро-упражнений с фильтром по сферам
- **Мосты** — связи между записями о еде, спорте и мыслях
- **Профиль** — статистика и «Банк ресурсов»
- Интерактивные сценарии для еды, спорта, мыслей и микро-вызовов

## Структура

```
src/
  components/   переиспользуемые компоненты (Triangle, карточки, таб-бар)
  flows/        интерактивные сценарии практик
  pages/        экраны (онбординг, today, practices, bridges, profile)
  context/      глобальный стейт приложения
  data/         мок-данные
  styles/       глобальные стили и переменные
  types/        типы
```
